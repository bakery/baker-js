Bootstrap your project
=======

```
npm install
cd test
ln -s ../app/scripts scripts
ln -s ../app/templates templates
cd ..
```

Run the app (http://localhost:9001)
=======

```
grunt 
```

Test (http://localhost:9001)
=======
```
grunt test
```

Dependencies
=======
[Grunt.js 0.4+](http://gruntjs.com/)




Happy baking!
